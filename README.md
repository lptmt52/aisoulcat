**aisoulcat** 是一个开源的 AI 编程助手，致力于提供 Cursor 的开源替代方案。本项目是对 [Void](https://github.com/voideditor/void) 的二次开发。

在您的代码库中使用 AI 代理，检查并可视化变更，并可以在本地带入任何模型或主机。aisoulcat 直接将消息发送给提供商，而不会保留您的数据。

本仓库包含 aisoulcat 的完整源代码。如果您是新用户，欢迎加入！

- 👋 [Discord (原项目)](https://discord.gg/RSNjgaugJs)

## 关于此分支

本项目是从 Void 派生（Fork）而来的，目前被重新命名为 **aisoulcat**。我们正在此基础上进行进一步的定制和开发。

## 说明

Void IDE 开发团队目前已暂停该仓库的工作，以探索一些新颖的编码理念。原项目仍可运行，但随着时间的推移，由于缺乏维护，某些现有功能可能会停止工作。

在开发暂停期间，原作者可能不会积极审查 Issue 和 PR，但他们会回应通过 [邮件](mailto:hello@voideditor.com) 发送的关于构建和维护您自己的 Void 版本的查询。

## 参考

aisoulcat 是 [vscode](https://github.com/microsoft/vscode) 仓库的一个分支的分支。有关代码库的指南，请参阅 [VOID_CODEBASE_GUIDE](./VOID_CODEBASE_GUIDE.md)。

有关如何开发您自己的 aisoulcat/Void 版本的指南，请参阅 [HOW_TO_CONTRIBUTE](./HOW_TO_CONTRIBUTE.md) 以及 [void-builder](https://github.com/voideditor/void-builder)。
